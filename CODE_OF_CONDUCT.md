# Code of Conduct

## Our standard

GoldBerry is a framework project about knowledge, interpretation, and critique. That makes disagreement normal. The standard here is not “agree with the framework”; it is “engage rigorously and respectfully”.

Participants are expected to:
- criticise ideas without demeaning people
- distinguish evidence from assertion
- avoid harassment, intimidation, or pile-ons
- avoid discriminatory language or targeted abuse
- stay open to correction, including correction of the framework itself

## Unacceptable behaviour

Examples include:
- personal attacks
- harassment or stalking
- hate speech
- sustained bad-faith disruption
- publishing private information without consent
- using the project space to target protected groups or vulnerable communities

## Project-specific norm

Because GoldBerry addresses power, marginalisation, and knowledge hierarchies, contributors should take extra care not to use those topics as cover for rhetorical aggression.

## Enforcement

Maintainers may remove comments, close threads, or block contributors whose behaviour undermines safe and serious participation.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project-managed community spaces.
